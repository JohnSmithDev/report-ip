Your public IP address is {{ ip_addr }} What is this?

An "Internet Protocol" (IP) address is like a postal or zip code on the internet. It can be used to identify you or other people using the same internet connection as you. Learn more at the Wikipedia page on this subject.

Where does Google think you are located? What is this?

Various companies, including Google, have built up databases that map IP addresses to physical real-world locations. These are not always accurate - if a latitude and longitude value is in the database, you can see for yourself how accurate the details are for your current IP address. Learn more at the Wikipedia page on this subject.

Request headers What is this?

Every time your web browser fetches some information - e.g. a web page or a picture - from the internet, it also provides information about you, your browser or your computer. This can include things like what your preferred language is and what type of browser you have. Learn more at the Wikipedia page on this subject.
